# Domain-Name-Generator
Create a script that generate all the possible domain name combinations from a list of pronouns, adjectives and nouns.

For example:

 - let pronoun = ['the','our'];
 - let adj = ['great', 'big' ];
 - let noun = ['jogger','racoon'];

Should generate something like:

 - thegreatjogger.com
 - thegreatracoon.com
 - ourgreatjogger.com
 - ourgreatracoon.com
 - thebigjogger.com
 - thebigracoon.com
 - ourbigjogger.com
 - ourbigracoon.com
